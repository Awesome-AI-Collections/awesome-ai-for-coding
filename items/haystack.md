# Haystack

## 一句话总结

一个面向 RAG、搜索与 Agent 应用的开源框架，适合搭建可检索问答和知识工作流。

## 核心场景

- 面向 RAG、搜索与 Agent 应用的开源框架
- 适合开发可检索问答、知识工作流和带文档理解能力的系统

## 为什么值得关注

- 开源可自建，方便私有部署和二次开发。
- 能与 OCR、抽取、RAG、客服或业务系统继续串联。
