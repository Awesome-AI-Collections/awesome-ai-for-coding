# dmux

## 一句话总结

一个基于 Git worktree 的开发环境复用器，适合并行 AI 开发时快速切换隔离环境。

## 核心场景

- 多分支并行开发
- 复用本地环境而不是重复搭建
- 让不同 Agent 在隔离上下文里工作

## 为什么值得关注

- 解决的是“环境切换”这个真实痛点
- 对 worktree 流派和多 Agent 开发特别友好
