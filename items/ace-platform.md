# ACE Platform

## 一句话总结

一个智能体控制环境平台，强调更强的安全监控与受控执行。

## 核心场景

- 给 Agent 提供可控执行环境
- 做安全监控和行为约束
- 降低高权限 Agent 带来的风险

## 为什么值得关注

- 把“安全边界”放在了平台层处理
- 对生产环境的 Agent 托管很重要
