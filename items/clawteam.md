# ClawTeam

## 一句话总结

港大开源的多智能体协同框架，用来让多个 AI 自动分工协作完成复杂任务。

## 核心场景

- 复杂任务拆解与分工
- 多 Agent 协作执行
- 用团队化方式提升单 Agent 容错

## 为什么值得关注

- 是典型的“AI 团队”框架
- 很适合需要显式分工协作的任务
