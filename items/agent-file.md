# agent-file

## 一句话总结

一个开源 Agent 文件格式，用于把记忆、行为和状态打包成可移植容器。

## 核心场景

- 在项目间迁移 Agent
- 做 Agent 版本控制与复用
- 让 Agent 状态不再依赖单一运行环境

## 为什么值得关注

- 它解决的是 Agent “可移植性”问题
- 对跨项目复用和团队协作都很关键
