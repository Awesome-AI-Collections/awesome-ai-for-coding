# supabase-mcp

## 一句话总结

一个把 Supabase 的数据库、认证与边缘函数能力接给 Claude 的 MCP 服务器。

## 核心场景

- 让 Claude 直接访问 Supabase
- 在一个通道里管理数据库、Auth 和 Edge Functions
- 提升全栈 AI 开发效率

## 为什么值得关注

- Supabase 是 AI 产品常见后端选项
- MCP 化之后更适合 Agent 直接使用
