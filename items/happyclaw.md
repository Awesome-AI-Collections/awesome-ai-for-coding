# HappyClaw

一个基于 Claude Code 运行时构建的自托管多用户 AI Agent 系统，支持 Web 与多种 IM 渠道接入。

## 适合谁看

- 开发者、技术团队、自托管用户

## 官方链接

- **GitHub:** https://github.com/riba2534/happyclaw
- **更新记录:** https://github.com/riba2534/happyclaw/releases

## 标签

- `Claude Code`, `编程 Agent`, `自托管`, `多用户`, `IM 集成`

## 说明

HappyClaw 适合希望把 Claude Code 作为正式生产力工具接入团队协作的用户。它强调自托管、多用户隔离、消息渠道统一接入，以及基于 Claude Agent SDK 的原生运行时复用。除 Web 界面外，还支持飞书、Telegram、QQ、钉钉、微信等渠道，并提供文件读写、终端操作、浏览器自动化、定时任务、记忆系统和多提供商负载均衡等能力。
