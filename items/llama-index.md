# LlamaIndex

## 一句话总结

一个面向 RAG 与知识索引的开发框架，适合把文档、数据库和工具接入 LLM 应用。

## 核心场景

- 面向 RAG 与知识索引的开发框架，强调数据连接、检索与问答链路
- 适合把文档、数据库、API 和工具系统接入 LLM 应用

## 为什么值得关注

- 开源可自建，方便私有部署和二次开发。
- 能与 OCR、抽取、RAG、客服或业务系统继续串联。
