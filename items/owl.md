# owl

## 一句话总结

camel-ai 开源的多智能体协作框架，可让专业化 Agent 借助浏览器、终端、函数和 MCP 进行团队作战。

## 核心场景

- 复杂研发任务拆解
- 多 Agent 团队式执行
- 把浏览器、终端和工具调用组合起来

## 为什么值得关注

- 是近期热度很高的多 Agent 协作框架
- 非常强调“Agent 军团”式工作方式
