# AutoGen

## 一句话总结

微软开源的多智能体开发框架，适合搭建可编排的 Agent 工作流与自动化系统。

## 核心场景

- 多智能体开发框架，适合组织多个 Agent 分工协作完成复杂任务
- 适合用于构建自动研究、工具调用、任务编排和长流程执行系统

## 为什么值得关注

- 开源可自建，方便私有部署和二次开发。
- 能与 OCR、抽取、RAG、客服或业务系统继续串联。
