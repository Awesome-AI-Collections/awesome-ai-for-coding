# Langfuse

## 一句话总结

一个开源 LLM 工程平台，提供 tracing、评测、prompt 管理和调试能力，适合 AI 应用团队把实验阶段的 Agent 工作流推进到生产环境。

## 核心场景

- 追踪 LLM 调用、Agent 行为、检索链路和用户会话
- 管理 prompt、数据集和评测流程，做持续迭代
- 为生产中的 AI 应用提供观测、调试和协作基础设施

## 为什么值得关注

- 它已经不是单点 tracing 工具，而是逐步长成一整套 LLM 工程工作台
- 自托管和生态集成都很成熟，适合真正要上线 AI 功能的团队
- release 节奏非常快，说明项目仍在高频演进
