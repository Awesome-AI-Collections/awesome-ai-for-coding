# crewAI

## 一句话总结

一个面向任务分工与角色协作的多智能体框架，适合搭建可复用的 Agent 工作流。

## 核心场景

- 多智能体协作框架，强调角色分工、任务拆解和工作流编排
- 适合构建需要多个 Agent 协同推进的自动化系统与开发原型

## 为什么值得关注

- 开源可自建，方便私有部署和二次开发。
- 能与 OCR、抽取、RAG、客服或业务系统继续串联。
