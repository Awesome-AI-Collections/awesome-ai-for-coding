# n8n-as-code

## 一句话总结

一个把 n8n 低代码工作流转成代码管理的工具，便于版本控制和长期维护。

## 核心场景

- 管理 n8n 工作流版本
- 把低代码流程纳入代码审查和协作
- 减少工作流配置漂移

## 为什么值得关注

- 解决了低代码工作流难维护的问题
- 对自动化团队的可维护性很有价值
