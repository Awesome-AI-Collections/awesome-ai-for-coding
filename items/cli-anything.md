# CLI-Anything

## 一句话总结

一个把任意工具或 API 转成 Claude CLI 可调用接口的框架，方便 Agent 直接操控软件。

## 核心场景

- 把现有软件快速接给 Claude 或其他 Agent
- 减少手写胶水层和繁琐集成
- 为桌面软件或本地工具增加 Agent 控制能力

## 为什么值得关注

- 抓住了“任何工具都能秒变 Agent 工具”这个需求
- 对 AI 软件操控和本地工具接入很有启发性
