# OpenLogs

## 一句话总结

一个面向 AI 智能体的可观测性平台，用于实时监控行为日志和运行轨迹。

## 核心场景

- 监控 Agent 行为日志
- 追踪执行路径与异常行为
- 为多 Agent 系统提供可观测性基础设施

## 为什么值得关注

- Agent 真正落地后，日志与可观测性会变得非常重要
- 它对应的是 AI 系统的“监控层”
