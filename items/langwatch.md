# LangWatch

## 一句话总结

一个面向 LLM 应用评测、仿真测试和生产观测的开源平台，适合 AI 工程团队系统化提升 Agent 的可靠性、性能和回归测试能力。

## 核心场景

- 在发布前对 Agent 做端到端仿真和场景测试
- 把 tracing、dataset、evaluation 和 prompt 优化串成闭环
- 在生产环境持续监控 AI 应用表现并定位回归问题

## 为什么值得关注

- 它比单纯 observability 更进一步，把 agent simulation 和 evaluation 也做进来了
- 对“上线后不想靠人工盯”的 AI 团队特别有价值
- 2026 年 4 月仍在高频发布，活跃度非常强
