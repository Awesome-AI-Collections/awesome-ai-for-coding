# mcp-use

## 一句话总结

一个让 MCP 工具链和服务器更易即插即用的套件，用于多智能体系统的标准化工具集成。

## 核心场景

- 快速接入 MCP 工具
- 减少手搓工具桥接层
- 为多 Agent 系统建立统一工具通道

## 为什么值得关注

- 把 MCP 从“黑魔法”变成可复用基础设施
- 很适合正在补工具层的 Agent 团队
