# claude-squad

## 一句话总结

一个帮助你管理 Claude 小分队的多会话协作工具，适合统一调度多个 Claude 实例。

## 核心场景

- 管理多 Claude 会话
- 协调不同 Agent 的分工
- 让一组 Claude 按团队方式协同

## 为什么值得关注

- 非常适合“AI 团队指挥官”式工作流
- 能显著降低多实例管理成本
